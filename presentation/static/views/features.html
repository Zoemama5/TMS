Features Overview

A clear, reliable, and secure authentication system built with Node.js, JWT, and Google OAuth — following clean architecture principles.

User Registration

Create new accounts with bcrypt password hashing and email verification to prevent duplicates.

JWT Authentication

Generate and validate secure JSON Web Tokens for stateless user sessions.

Password Reset

Implement secure password recovery with token-based email links and expiry validation.

Google OAuth 2.0

Allow sign-in with Google for convenience, using verified OAuth tokens and scopes.

CSRF Protection

Protect against unauthorized actions by verifying CSRF tokens in requests.

Security Best Practices

Follows environment-based configs, token expiry, and input sanitization.

Scalable API

Designed with modular endpoints and service layers for maintainable, scalable APIs.

Email Integration

Sends verification and reset links using secure SMTP or NodeMailer configurations.

Clean Architecture

Implements service and controller separation, ensuring organized and testable code.
